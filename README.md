- `Descrição do Projeto: Sorteio de Amigo Secreto`

Este projeto em JavaScript é uma aplicação simples e funcional para realizar sorteios de Amigo Secreto. Desenvolvido como parte de um desafio proposto pelo ONE (Oracle Next Education) em parceria com a Alura, o programa oferece uma experiência intuitiva para o usuário, permitindo adicionar participantes e realizar o sorteio de forma prática.

# :hammer: Funcionalidades:

- `1. Adicionar Participantes:` Os nomes dos participantes podem ser inseridos individualmente por meio de um campo de entrada. Ao clicar no botão "Adicionar", os nomes são armazenados em uma lista.


- `2. Realizar o Sorteio:` Após inserir todos os nomes, basta clicar no botão "Sortear Amigos" para que o programa escolha aleatoriamente um nome da lista, garantindo um sorteio justo e divertido.


- `3. Interface Simples:` A interface do programa é amigável, com botões claros e funcionalidades que facilitam o uso.



# :hammer: Objetivo:

- `Este projeto busca reforçar os conhecimentos em:`

Manipulação do DOM (Document Object Model);

Eventos em JavaScript;

Lógica de programação aplicada ao sorteio de elementos.


# :hammer:Tecnologias Utilizadas:

- `HTML:` Estrutura do programa.

- `CSS:` Estilo e design.

- `JavaScript:` Lógica e interação dinâmica.


O desafio propõe uma abordagem prática e criativa para o desenvolvimento de soluções interativas, incentivando o aprendizado e a aplicação dos conceitos fundamentais de programação.
